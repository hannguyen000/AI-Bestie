# AI-Bestie
This is a mobile application designed for a target audience of Gen Z users, especially young women. The app features a multi-section UI with a soft pastel color palette to create a friendly and aesthetically pleasing experience.
The core functionality revolves around AI characters with distinct personalities, categorized into three different types. These AI companions are designed primarily for chatting, acting like a “bestie” who can offer advice, emotional support, and care for the user’s mental well-being.
In addition to the AI chat feature, the app includes various functionalities tailored to users’ interests, such as a mood tracker, period tracker, skincare tracking, water intake reminders, and fashion advice.
